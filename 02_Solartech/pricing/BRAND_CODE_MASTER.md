## SKU_Display Generation Rules

### Priority Order:
1. **If `brand` field populated** ‚Üí Use brand code from this list
2. **Else if SKU prefix = `COMP-`** ‚Üí Use `WCI`
3. **Else if SKU prefix = `SMFG-`** ‚Üí Use `STL`
4. **Else if SKU is numeric only** ‚Üí Use `KIN`
5. **Else if SKU prefix = `D-`** ‚Üí Skip (duplicate)
6. **Else** ‚Üí Leave brand portion blank

### Formula Logic (Excel):
```
=IF(brand<>"", VLOOKUP(brand, BrandCodes, 2, FALSE),
  IF(LEFT(SKU,5)="COMP-", "WCI",
  IF(LEFT(SKU,5)="SMFG-", "STL",
  IF(ISNUMBER(VALUE(SKU)), "KIN",
  ""))))
```
